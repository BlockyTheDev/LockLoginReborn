# Security Policy

## Supported Versions

The following version of LockLogin are
currently supported

### NS = Non release ( Snapshot - Release Candidate - Support build )

|     Version     |         Supported        |
| ----------------|--------------------------|
|     1.1.0.9     | :white_check_mark:       |
|     1.12.1      | :white_check_mark:       |
|     NS 1.12.1   | :white_check_mark:       |
|     < 1.1.0.8   | :x:                      |
